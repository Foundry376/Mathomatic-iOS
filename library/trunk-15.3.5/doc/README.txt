
This directory contains the complete Mathomatic user documentation in HTML.
To read or print this documentation, point your web browser to the file
"index.html". If the PDF documentation was created, it is in the file
"manual.pdf". This PDF document is made by the htmldoc program by typing
"make pdf" in the parent directory.

When copying the Mathomatic documentation, please copy this entire doc
directory, not just selected HTML files from it.

To view online the most recent documentation, visit
http://mathomatic.org/math/doc/ with your web browser.
